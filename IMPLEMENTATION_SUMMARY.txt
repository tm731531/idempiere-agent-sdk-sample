連接失敗?        → 檢查 .env 中的 DB_HOST, DB_USER
API 錯誤?        → 確認 ANTHROPIC_API_KEY 已設置
Port 5000 占用?  → 使用 `lsof -i :5000` 找出程序

詳見: SETUP_GUIDE.md

💡 獲取 API Key:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 訪問: https://console.anthropic.com
2. 建立賬戶或登錄
3. 進入「API Keys」分頁
4. 複製 API Key
5. 粘貼到 .env: ANTHROPIC_API_KEY=sk-ant-...
